Advanced Download

For a minimal footprint you can download just the Cycle2 "core". This provides a flexible slideshow engine with minimal bells and whistles. Cycle2 core supports delarative, responsive, automatic slideshows and includes the fade, fadeout, and scrollHorz transition effects.

All of the core plugins listed below are included in the main jquery.cycle2.js distribution files (production and development). If there are features you don't want or need you can pick and choose which plugins to include on your page.

If you're not sure which files to download, keep it simple and just click the first button on the main download page.
