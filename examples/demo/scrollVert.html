Scroll Vert

Cycle2's scrollVert transition plugin must be included in order to use the scrollVert effect. See the Download page for a link to download the scrollVert plugin.

The scrollVert effect is achieved by assignin the fx option to scrollVert.

Simple

<script src="http://ajax.googleapis.com/ajax/libs/jquery/1/jquery.min.js"></script>
<script src="http://jquery.malsup.com/cycle2/jquery.cycle2.js"></script>
<script src="http://jquery.malsup.com/cycle2/jquery.cycle2.scrollVert.js"></script>

...

<div class="cycle-slideshow"
    data-cycle-fx="scrollVert"
    >
    <img src="http://jquery.malsup.com/cycle2/images/p1.jpg">
